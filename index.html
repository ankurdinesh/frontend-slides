What this repo is

A toolkit for authoring shareable HTML presentations.

  • Each presentation is a single HTML document with inline CSS and JavaScript.
  • The layout rules enforce full-viewport slides with no internal scrolling.
  • Helper scripts handle PDF export, PowerPoint extraction, and deployment.
  • You preview by opening the HTML file, not by running a dev server.
02

Next actions

Use this file as your starting point.

Customize content

Replace the copy, adjust theme variables, and add or remove slides directly in `index.html`.

Export to PDF

Run `bash scripts/export-pdf.sh ./index.html` when you need a static handout.

Deploy to a URL

Run `bash scripts/deploy.sh ./index.html` when you want a shareable hosted version.
